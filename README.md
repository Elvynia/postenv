# Linux install
## Remote one time install
```bash
curl -fsSL https://raw.githubusercontent.com/Elvynia/postenv/main/linux/bootstrap/remote.sh | bash \
    && . ~/.bashrc
```
## Local install to make changes
```bash
curl -fsSL https://raw.githubusercontent.com/Elvynia/postenv/main/linux/bootstrap/local.sh | bash \
    && . ~/.bashrc
```
